## 🔧 n8n工作流实现方案

### **工作流1：主调度工作流**
```json
{
  "name": "AI选股系统主调度",
  "nodes": [
    {
      "name": "定时触发器",
      "type": "n8n-nodes-base.scheduleTrigger",
      "parameters": {
        "rule": {
          "interval": [
            {
              "field": "hours",
              "minutesInterval": 1
            }
          ]
        }
      }
    },
    {
      "name": "数据采集模块",
      "type": "n8n-nodes-base.httpRequest",
      "parameters": {
        "url": "http://localhost:3000/api/data-collection"
      }
    },
    {
      "name": "AI分析模块",
      "type": "n8n-nodes-base.httpRequest",
      "parameters": {
        "url": "http://localhost:3000/api/ai-analysis"

      "name": "决策执行模块",
```

## 🔗 技术栈

### **n8n节点类型：**
1. **HTTP Request** - 调用外部API
2. **Python** - 运行Python脚本
3. **PostgreSQL** - 数据库操作
4. **Schedule Trigger** - 定时任务
5. **Function** - JavaScript代码执行
6. **Webhook** - 接收外部请求
