## 二、API 端点清单 (31个)

### 宏观数据 (6个)
| 端点 | 数据 | 缓存TTL |
|------|------|---------|
| `GET /api/data/global-indices` | 美/中/港/欧/日 指数 | 60s |
| `GET /api/data/forex` | DXY/EURUSD/USDJPY/GBPUSD/USDCNY | 30s |
| `GET /api/data/commodities` | 黄金/白银/原油/铜/锂/大豆 | 60s |
| `GET /api/data/bond-yields` | 美/中/日/德国债收益率 | 60s |
| `GET /api/data/economic-calendar` | 当日经济事件(含预期/前值) | 300s |
| `GET /api/data/financial` | HS300财务/行业利润增速 | 300s |

### 行情数据 (7个)
| 端点 | 数据 | 缓存TTL |
|------|------|---------|
| `GET /api/data/market` | A股指数/成交量/北向/情绪/涨跌比 | 30s |
| `GET /api/data/us-market` | 美股指数/VIX/板块/盘前期指 | 60s |
| `GET /api/data/hk-market` | 恒指/恒科/南向/腾讯/美团/阿里 | 60s |
| `GET /api/data/northbound-flow` | 北向净流入/沪股通/深股通 | 60s |
| `GET /api/data/sector-flow` | 8大板块资金流向 | 120s |
| `GET /api/market/hotspots` | 热点板块+领涨股 | 120s |
| `GET /api/market/capital-flow` | 资金流向排行 | 120s |

### 情报/新闻 (5个)
| 端点 | 数据 | 缓存TTL |
|------|------|---------|
| `GET /api/data/intel` | 智库情报(政策/公告/研报/地缘) | 120s |
| `GET /api/data/rss-news` | 全球资讯聚合(RSS) | 300s |
| `GET /api/data/central-bank` | 央行利率/会议/信号 | 300s |
| `GET /api/intel/overview` | 情报概览+风险等级 | 300s |
| `GET /api/intel/analyze` | 情报分析+风险预警 | 600s |

### N8N 工作流集成 (10个)
| 端点 | 用途 | 被N8N工作流调用 |
|------|------|----------------|
| `GET /api/watchlist` | 监控股票列表 | 智能选股/监控 |
| `GET /api/watchlist/anomalies` | 异常波动检测 | 盘中异动监控 |
| `GET /api/risk/stock/<code>` | 个股风险扫描 | 持仓风险监控 |
| `GET /api/market/hotspots` | 热点板块 | 选股分析 |
| `GET /api/market/capital-flow` | 资金流向 | 资金分析 |
| `GET /api/market/sentiment` | 市场情绪 | 综合判断 |
| `GET /api/market/hot-industries` | 热门行业 | 行业轮动 |
| `GET /api/market/ai-summary` | AI市场总结 | 收盘报告 |
| `GET /api/market/global-indices` | 全球指数 | 全球监控 |
| `POST /api/webhook/n8n-result` | 接收N8N结果 | 结果写入日志 |

### 技能/系统 (3个)
| 端点 | 用途 |
|------|------|
| `GET /api/skills` | 技能列表 |
| `POST /api/skills/run` | 执行AI技能 |
| `GET /api/data/think-tank-pipeline` | 智库中枢流程代理 |
| `GET /api/data/think-tank-digest` | 智库日报代理 |
| `GET /api/data/think-tank-stats` | 智库统计代理 |

## 六、启动与运维

```bash

cd C:\Users\XIAOMI\Desktop\ai_stock
python app.py

# 健康检查
curl http://localhost:4000/api/data/market

# N8N 投递结果

  -H "Content-Type: application/json" \
  -d '{"type":"选股结果","stocks":[...]}'

ls ai_stock/logs/n8n_result_*.jsonl
```
