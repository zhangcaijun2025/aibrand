# AiBrand 后端核心技术模块 简报

> 2026-06-04 — 供产品设计与前端开发参考

---

## 一句话

AiBrand 后端已建成 **100+ API 的全域 AI 运营平台**，核心差异化模块是 **Agent 感知层**（问候·进化·对话·精灵）。

---

## 十大模块速览

| # | 模块 | 核心能力 | API 数 |
|---|------|---------|:--:|
| 1 | **Agent 感知层** | 问候(时间/记忆/状态) · 进化引擎(学习偏好) · 8个预设Agent · SSE对话 | 28 |
| 2 | **AI 服务** | Dify RAG(知识库检索) · n8n(24工作流) · LangChain Bridge(7端点) | 20+ |
| 3 | **用户认证** | 邮箱/手机/Google登录 · JWT 鉴权 · API Key管理 | 15+ |
| 4 | **多平台账号** | 12平台 OAuth · 分组管理 · 统计 | 15+ |
| 5 | **订阅积分** | 3订阅计划 · 积分充值 · 配额管理 | 8+ |
| 6 | **通知系统** | 消息列表 · 未读计数 · 已读标记 | 4 |
| 7 | **内容素材** | 素材CRUD · 媒体库 · 分组管理 | 10+ |
| 8 | **发布引擎** | 多平台一键发布 · 状态追踪 | 5+ |
| 9 | **数据分析** | 跨平台数据 · 互动统计 | 5+ |
| 10 | **部署架构** | Docker 17容器 · Nginx · MongoDB · Redis | - |
