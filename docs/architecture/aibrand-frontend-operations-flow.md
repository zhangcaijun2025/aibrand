## 三、一键分发 Tab — 内容发布流程

```
┌──────────────────────────────────────────────────────────────────────┐
│                       5 阶段发布管线                                   │
│                                                                      │
│  Phase 1: EDIT (编辑)                                                │
│  ┌────────────────────────────────────────────────────┐              │
│  │  标题输入 → 正文编辑 → 标签输入                       │              │

│  │  [下一步：质检审查] ──────────────┐                   │              │
│  └───────────────────────────────────┼───────────────────┘              │
│                                      │                                  │
│  Phase 2: CHECKING (质检)            ▼                                  │
│  ┌────────────────────────────────────────────────────┐              │
│  │  4 维度流式评分 (400ms/维度):                        │              │
│  │  ┌──────────┬──────────┬──────────┬──────────┐    │              │
│  │  │ 内容质量  │ GEO优化  │ 平台合规  │ 原创度   │    │              │
│  │  │  85分 ✅  │  82分 ✅  │  90分 ✅  │  88分 ✅  │    │              │

│  └────────────────────────────────────────────────────┘              │
│                                      │                                  │
│  Phase 3: CONFIRM (确认)             ▼                                  │
│  ┌────────────────────────────────────────────────────┐              │
│  │  质检通过 · 85分                                     │              │
│  │  内容预览 (标题/正文/标签)                             │              │

│  │  [取消]  [确认发送] ────────────────┐                │              │
│  └─────────────────────────────────────┼────────────────┘              │
│                                        │                               │
│  Phase 4: SENDING (发送)               ▼                               │
│  ┌────────────────────────────────────────────────────┐              │
```
