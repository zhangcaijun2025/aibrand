1.2 一人公司场景特别建议

  • 不要按座位定价 — 你的用户就是一个人
  • 用量限额 + 功能升级组合 — 基础版按用量限制，Pro 版解锁高级功能+更高用量

    • 二、定价心理学：锚定、诱饵与价格敏感度

    🎯 价格锚定效应 (#1 心理杠杆)
    用户看到的第一价格成为心理基准。实战：将 Enterprise /“联系我们”放在定价页最左边，让 Pro 计划显得合理。中国 AI 标准锚点：68元/月（个人版）、200元/月（专业版）、500元/月（企业版）。一人公司第一锚点 ¥49-99/月，第二锚点 ¥149-199/月。
    ⚖️ 诱饵效应 (Decoy Effect)
    3-4 层定价最优，5+ 层导致分析瘫痪。诱饵应让目标层看起来不可拒绝。⚠️ 对抗性验证：学术研究表明诱饵效应对中间层推动作用很弱（仅约1%偏好偏移），在 B2B 场景更弱。

    3.3 中国定价实战建议

  • 必须有免费层，用用量上限控制
  • 积分制优先 — 100积分=¥1 更“中国化”
  • 年付省 2 个月 (买10送2) 话术 > 百分比
  • 双轨制 — 国内版低价 + 海外版高价

    六、一人公司 AI SaaS 定价实操指南

    6.1 定价决策矩阵

  • 固定推理成本 → 混合定价：基础订阅 ¥49-99/月（含N次调用）+ 超额按量
  • 可变推理成本 → 积分制：¥0.07/积分
  • 几乎零边际成本 → 纯订阅制 + 用量上限
  • 高额边际成本 → 按量付费为主 + 可选订阅包月

    • 6.2 三步定价方程式

    Step 1 锚点：中国 ¥49-99/月 (个人) / ¥149-299/月 (专业)。海外 $29-49 / $79-149。
    Step 2 三级结构：免费版 (用量耗尽触发) → 专业版 (主力，支付意愿60-70%) → 企业版 (锚点)。
    Step 3 转化路径：主页 → 免费注册 → 首次价值感(5min内) → 用量追踪 → 80%提醒 → 升级弹窗。

    6.3 十大金律

    🎯 不要按座位定价
    ⚓ Enterprise 层做价格锚
    🆓 免费层+用量上限
    📊 14天试用 > 30天
    💰 年付优惠 ≥25%
    🔢 3-4层定价
    🇨🇳 积分 > 订阅 (中国)
    🚫 不要从$20起锚
    📈 A/B测试定价页
    ⚡ 用量耗尽是最佳触发
