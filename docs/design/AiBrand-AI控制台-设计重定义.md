# AiBrand AI 控制台 — 设计重定义

## 当前问题

```
❌ 静态卡片网格 — 像 CRM 后台，不像 AI 平台
❌ 聊天框在角落 — Agent 应该是视觉中心，不是附属品
❌ Tab 切换 — 割裂体验，AI 应该主动呈现而非被动等待点击
❌ 浅色主题 — 没有 AI 的未来感和沉浸感
❌ 按钮驱动 — 应该是意图驱动（说出需求→系统理解→自动执行）
❌ 无动画 — 数据是"死"的，没有"系统在呼吸"的感觉
```

## AI 原生产品的六大特质

| 特质 | 描述 | 如何体现 |
|------|------|------|
| **1. 环境智能** | 信息不是卡片堆砌，而是自然流动、有机呈现 | 粒子背景、数据涟漪、实时脉冲 |
| **2. Agent 中心** | 对话是主要交互方式，不是辅助工具 | Agent 占据视觉重心，自然语言驱动一切 |
| **3. 生命感** | 系统状态不是 ON/OFF，而是"呼吸/心跳" | 进化指数动态变化、服务脉搏、微动画 |
| **4. 预判呈现** | 不等你找，主动告诉你需要知道的事 | 异常自动浮出、智能建议、动态优先级 |
| **5. 沉浸感** | 深色基底 + 生物荧光点缀 + 模糊玻璃 | 参考：Arc Browser、VS Code 暗色主题、Apple VisionOS |
| **6. 无摩擦** | 不打断思维流，操作如呼吸般自然 | Command+K 命令面板、智能路由、零点击预测 |

## 新设计方向

### 视觉语言
```
主题: 深空暗色 (Dark Cosmic)
基色: #080B12 (深空黑) → #131828 (面板) → #1A2332 (卡片)
荧光: 进化绿 #00F5A0 / 数据蓝 #4DABF7 / 警告琥珀 #FFB347 / 自愈紫 #A78BFA
材质: 磨砂玻璃 backdrop-blur + 微边框 border-white/5
动效: 呼吸脉冲 / 粒子漂浮 / 辉光渐变 / 平滑过渡
字体: Inter / JetBrains Mono (数据)
```

## 关键交互

1. **Agent 是主角** — 占据 60% 视觉面积，不是角落的聊天框
2. **Command+K** — 全局命令面板，键盘驱动，不依赖鼠标
3. **脉搏指示器** — 7个服务不是静态卡片，而是跳动的小光点
4. **进化指数** — 动画进度环，不是数字
5. **智能建议** — Agent 主动说"你上次的竞品分析还没完成，要继续吗？"
6. **粒子背景** — 根据系统负载动态变化（空闲=少量、繁忙=密集）
7. **语音优先** — 输入框旁边有麦克风按钮

---

> **一句话**: 用户坐下来的第一感受应该是"我面对的是一个活的 AI 系统"，而不是"又一个后台管理系统"。
