## 一、产品概述

### 1.1 产品定位

**AiBrand** 是一站式 AI 全域运营平台。帮助超级个体、一人公司、中小微企业用 AI 完成内容创作→多平台发布→客户互动→数据分析的完整闭环，实现品牌全球化运营。

**一句话描述**：用 AI 把一个人变成一个全球化运营团队。

### 1.2 目标用户画像

| 用户类型 | 特征 | 核心需求 |
|---------|------|---------|
| **超级个体** | 自媒体人、KOL、独立创业者 | 高效创作→一键多平台发布→节省时间 |
| **一人公司** | Solo-founder、自由职业者 | AI 替代团队、客户管理、品牌曝光 |
| **小微团队** | 5-20 人创业公司 | 协作内容工厂、数据分析、全球化分发 |

### 1.3 商业模式

- **SaaS 订阅**：Free / Pro / Enterprise 三级订阅
- **积分充值**：AI 内容生成按 Token 消耗计费
- **私有化部署**：企业级本地部署买断

## 二、设计哲学

### 2.1 核心设计原则

1. **AI-Native（AI 原生）**：AI 不是功能，是架构。用户通过对话与 Agent 交互，而非操作表单
2. **WeChat-Style Mobile First**：底部 4Tab 主导航，类微信的极简交互模型
3. **Dark Universe（暗色宇宙）**：深色基调 + 紫青渐变品牌色，营造科技感与专业感
4. **Progressive Disclosure（渐进呈现）**：默认展示核心信息，点击展开详情
5. **Global-Ready（全球化就绪）**：i18n 覆盖、RTL 预留、多时区支持

### 2.2 品牌设计系统

```
品牌色：
  Purple: #7C3AED (主色) → #A78BFA (浅) → #5B21B6 (深)
  Cyan:   #06B6D4 (辅色) → #67E8F9 (浅) → #0891B2 (深)

渐变：
  主渐变: 135deg, #7C3AED → #06B6D4
  辉光紫: 0 0 20px rgba(124, 58, 237, 0.3)
  辉光青: 0 0 20px rgba(6, 182, 212, 0.3)

背景层级：
  Base:     #0F0B1A (最深)
  Elevated: #1A1430 (卡片/面板)
  Overlay:  rgba(26, 20, 48, 0.8) (浮层)

文字层级：
  Primary:   #F5F3FF (主体文字)
  Secondary: #A5A0B4 (辅助文字)
  Muted:     #6B6580 (弱化文字)

圆角: 8px (小) / 12px (中) / 16px (大) / 24px (特大)
阴影: 辉光风格，无传统投影
字体: Inter (英文) + 系统默认中文
```

## 四、核心页面详细说明

### 4.1 Agent Home（/agent）— 🔴 设计优先级最高

**设计目标**：用户登录后的第一屏，Agent 苏醒体验

**页面结构**：
```
┌─────────────────────────────────────┐
│          [Dark Universe BG]         │
│                                     │
│         ⭕ Agent 呼吸光点            │
│       (渐变球体, 呼吸动画)          │
│                                     │
│     ┌─────────────────────┐        │
│     │  早安，[用户名]       │        │
│     │  今天有 3 个平台待发布  │        │
│     │  你的品牌健康度：92%   │        │
│     └─────────────────────┘        │
│                                     │
│  ┌──────────┐  ┌──────────┐       │
│  │ 记忆卡片1  │  │ 记忆卡片2  │       │
│  │ 上次话题   │  │ 竞品动态   │       │
│  └──────────┘  └──────────┘       │
│                                     │
│  ┌──────────────────────────────┐  │
│  │ 💬 跟 Agent 说你想做什么...     │  │
│  │                         [发送] │  │
│  └──────────────────────────────┘  │
│                                     │
└─────────────────────────────────────┘
```

**交互要点**：
- Agent 苏醒动画：呼吸光点 → 睁眼线 → 问候语渐入 → 记忆卡片滑入
- 呼吸球持续脉冲动画（在线状态指示）
- Cmd+K / 点击呼吸球 → 命令面板弹出
- 对话输入框自动聚焦

### 4.2 AI Creator（/create）— 🔴 设计优先级最高

**设计目标**：AI 驱动的内容创作工作台

**6步内容工厂工作流**：
```
Step 1: 意图分析 ──→ Step 2: 策略研究 ──→ Step 3: 选题生成
                                            ↓
Step 6: 发布策略 ←── Step 5: 质量检测 ←── Step 4: 内容生成
```

**页面布局**（三栏式）：
```
┌──────────┬──────────────────────┬──────────┐
│ 流程面板   │    主内容区 (编辑器)    │ 预览面板   │
│          │                      │          │
│ [Step 1] │  ┌────────────────┐  │ 小红书预览 │
│ [Step 2] │  │ AI 编辑器       │  │          │
│ [Step 3] │  │ (Markdown)     │  │ 知乎预览  │
│ [Step 4]●│  │                │  │          │
│ [Step 5] │  │                │  │ 抖音预览  │
│ [Step 6] │  └────────────────┘  │          │
│          │                      │          │
│  状态指示  │  底部：字数/平台适配建议│  [一键发布] │
└──────────┴──────────────────────┴──────────┘
```

**交互要点**：
- 左侧流程可点击跳转、可折叠
- 中间编辑器支持 Markdown/富文本切换
- 右侧实时预览不同平台的渲染效果
- 底部 AI 建议条：字数提示、标签建议、图片比例建议

### 4.3 Unified Publish（/publish）— 🟡 设计优先级高

**设计目标**：一站式多平台发布中心

### 4.5 浏览器扩展集成

**扩展与 Web App 的关系**：
- Web App 是内容策略中心（创作+编排+分析）
- 浏览器扩展是执行终端（发布+互动+数据采集）
- 通过 `window.postMessage` 实时通信
- 扩展提供 30+ 平台发布 + 小红书/抖音深度互动

**扩展 UI（独立于 Web App）**：
- Popup 弹窗：快速发布入口
- Sidepanel 侧边栏：发布进度+标签管理
- Options 设置页：平台账号+信任域名

## 六、关键用户流程

### 6.1 首次使用流程
```
注册 → 邮箱验证 → Agent 苏醒 → 平台授权引导 → 首次AI创作 → 首次发布 → 查看结果
```

### 6.2 日常内容发布流程
```
打开 AiBrand → Agent 问候 + 建议 → "帮我写一篇关于[话题]的文章"
→ AI 生成 → 预览各平台效果 → 选择平台 → AI 适配格式 → 一键发布 → 回来看数据
```

### 6.3 深度互动流程（小红书/抖音）
```
看到待回复评论 → 点击进入客户互动 → 查看评论列表
→ AI 建议回复 → 一键回复/批量回复 → 点赞互动粉丝内容 → 查看互动数据
```

## 八、动画规范

| 场景 | 动画 | 时长 | 缓动 |
|------|------|------|------|
| 页面切换 | 淡入+上移 20px | 300ms | ease-out-expo |
| 卡片入场 | 交错淡入 (stagger 80ms) | 400ms | ease-out |
| Agent 苏醒 | 呼吸→睁眼→问候 | 2.5s | 自定义关键帧 |
| 加载状态 | 骨架屏 + 脉冲 | 循环 | - |
| 发布进度 | 进度条 + 平台逐个点亮 | 实时 | - |
| 数字跳动 | 计数动画 | 800ms | ease-out |

**动画属性限制**：仅使用 `transform`、`opacity`、`clip-path`、`filter` — 禁止动画 `width/height/top/left/margin/padding`
