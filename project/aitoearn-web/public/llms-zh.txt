# AiBrand - AI 驱动的内容创作与社交媒体管理平台

> 本文件遵循 llms.txt 标准 (https://llmstxt.org/)，帮助 AI 模型更好地理解我们的平台。

## 关于 AiBrand

AiBrand 是一个综合性的 AI 驱动平台，彻底革新了内容创作和社交媒体管理方式。我们赋能创作者、营销人员和企业简化内容工作流程，并最大化他们在多个社交媒体平台上的影响力。

## 核心功能

### 1. AI 内容创作
- 多平台内容生成（文字、图片、视频）
- AI 驱动的文案和标题生成
- 可定制的内容模板和提示词
- 批量内容创作能力

### 2. 社交媒体管理
- 多平台账号集成（Instagram、Twitter、LinkedIn、TikTok、Facebook、YouTube 等）
- 统一内容发布面板
- 定时发布和内容日历
- 跨平台营销活动管理

### 3. 任务市场
- 内容创作者任务市场
- 基于佣金的收益机会
- 内容协作功能
- 创作者网络和社区

### 4. 素材库
- 基于云的资产管理
- 图片、视频和文档存储
- 模板库和品牌工具包
- 媒体组织和搜索

### 5. 数据分析与洞察
- 跨平台表现追踪
- 互动指标和分析
- 内容优化建议
- ROI 测量工具

## 技术栈

- **前端**: Next.js 14, React 18, TypeScript
- **样式**: Tailwind CSS v4, shadcn/ui 组件库
- **状态管理**: Zustand 持久化存储
- **国际化**: i18next（英文、中文）
- **UI 组件**: Radix UI 基础组件
- **媒体处理**: Sharp, HTML2Canvas
- **实时功能**: WebSocket 支持

## 用户流程

### 内容创作流程
1. 登录/注册 → 控制台
2. 选择内容类型和模板
3. 使用 AI 辅助生成内容
4. 编辑和自定义内容
5. 安排或发布到平台
6. 监控表现

### 任务市场流程
1. 浏览可用任务
2. 申请任务或接受指派
3. 根据要求创作内容
4. 提交审核
5. 获批后赚取积分

### 账号管理流程
1. 连接社交媒体账号
2. 授权平台访问
3. 管理多个频道
4. 配置发布偏好
5. 监控账号健康

## 核心页面与路由

- `/` - 首页和着陆页
- `/tasks` - 任务市场
- `/accounts` - 已连接的社交媒体账号
- `/publish` - 内容创作和发布
- `/library` - 素材和资产库
- `/analytics` - 表现仪表板
- `/pricing` - 订阅套餐
- `/settings` - 用户设置和偏好
- `/profile` - 用户资料管理

## API 结构

平台使用 RESTful API 架构：
- 基础 URL: 通过 `NEXT_PUBLIC_API_URL` 配置
- 认证: 基于令牌（JWT）
- 频率限制: 按用户等级应用
- 端点遵循 RESTful 约定
